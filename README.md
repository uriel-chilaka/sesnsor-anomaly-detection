# 📊 Sensor Anomaly Detection System

A Python-based anomaly detection pipeline designed to identify irregular patterns in multi-sensor data streams.

---

## 🚀 Features

- 📈 Sensor data preprocessing
- 🤖 Autoencoder-based anomaly detection
- 🔍 Real-time anomaly flagging
- 📊 Visualization of sensor behavior
- Modular ML pipeline

---

## 🛠️ Tech Stack

- Python
- NumPy
- Pandas
- PyTorch
- Matplotlib

---

## 🧠 Problem Statement

Sensor systems in embedded environments often generate noisy data.  
This project explores how neural networks (autoencoders) can detect anomalies in real-time data streams.

---

## 🏗️ Pipeline

1. Data Collection
2. Data Normalization
3. Autoencoder Training
4. Reconstruction Error Calculation
5. Anomaly Classification
